# Online Store

Simple online store

View [Deploy](https://blick7.github.io/online-store/index.html)

[Reference](https://www.horizonhobby.com/)

# Installation

Clone this repo and npm install

```sh
npm i
```

# Usage

## Development server

```sh
npm start
```

Accessible at `localhost:8080`

## Production build

```sh
npm run build
```
